## Datos de acceso

Los datos de acceso deben se proporcionados por la institución con la que se esta realizando la conexión.

| Parametro   | Descripción                                  |
| ----------- | -------------------------------------------- |
| x-auth-id   | identificador de banco                       |
| x-auth-key  | llave unica generada por la institución      |
| ip.servidor | Es la IP publica o del VPN de la institución |
| puerto      | puerto de acceso a rest API                  |

## Encabezado de las peticiones

Toda petición debe incluir en el header los parametros para autenticar al usuario

| Parametro  |
| ---------- |
| x-auth-id  |
| x-auth-key |

Ejemplo en **c#** de petición con los parametros en el header.

```js
var client = new RestClient("/consulta/publicos");
client.Timeout = -1;
var request = new RestRequest(Method.GET);
request.AddHeader("x-auth-id", "99");
request.AddHeader("x-auth-key", "8E9AC3E8B77C5B79161E5D1B2A7C0D06");
IRestResponse response = client.Execute(request);
Console.WriteLine(response.Content);
```
